# Changelog

## 1.1.1
- 字符串库新增 `DirectSend` 选项：勾选后由字符串库直接发送，不再填入主窗体发送框
- 修复字符串库删除与行布局相关问题
- 修复 `CHAR-HEX` 工具重复打开与释放逻辑

## 1.1.0
- 首个可推送里程碑版本
- 完成 `.NET 8` 重构与接收管线拆分
- 统一串口 / TCP / UDP 的基础生命周期与状态管理
- 新增发送字符串库工具与 `CHAR-HEX` 扩展

## 1.0.3
- 发送字符串库条目支持直接编辑，并默认自动加载和关闭自动保存
- 扩展 `CHAR-HEX` 小工具，新增字符串与 HEX 字符串双向转换
- 发送字符串库保存格式统一为 JSON 文件

## 1.0.2
- 新增“发送字符串库”工具窗体：支持标签/字符串新增、行删除、HEX/字符串填充到主窗口发送框
- 新增导入/导出功能，保存并恢复发送字符串条目列表
- 继续收敛 `Form1` 的工具入口与配置生成逻辑

## 1.0.1
- 收敛 `Form1` 状态切换逻辑，移除后台线程同步读取 UI 的路径
- 串口重构：将接收改为可取消后台循环，并将关闭流程改为异步收敛，避免关闭卡死
- UDP/TCP 重构：统一生命周期和状态重置，补充注销能力并降低重复代码

## 1.0.0
- 原始稳定基线版本
