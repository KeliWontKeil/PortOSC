# Changelog

## 1.1.1 (Local)
- 更新版本管理体系为 `1.2.3`：
  - `1`：大版本
  - `2`：可推送小版本
  - `3`：本地修订号（每次本地提交 `+1`）
- 文档同步：`docs/AI_WORKFLOW.md`、`docs/RULES.md`、`.github/copilot-instructions.md`、`docs/PROJECT_DESCRIPTION.md`
- 连续重构：收敛 `Form1` 状态切换逻辑，移除后台线程同步读取 UI 的路径
- 串口重构：将接收改为可取消后台循环，并将关闭流程改为异步收敛，避免关闭卡死
- UDP/TCP 重构：统一生命周期和状态重置，补充注销能力并降低重复代码

## 1.1.2 (Local)
- 新增“发送字符串库”工具窗体：支持标签/字符串新增、行删除、HEX/字符串填充到主窗口发送框
- 新增 `.ison` 导入/导出功能，保存并恢复发送字符串条目列表
- 继续收敛 `Form1` 的工具入口与配置生成逻辑

## 1.1.3 (Local)
- 发送字符串库条目支持直接编辑，并默认自动加载和关闭自动保存
- 扩展 `CHAR-HEX` 小工具，新增字符串与 HEX 字符串双向转换
- 保持发送字符串库无需双击填充，仍通过按钮显式回填主窗口发送框

## 1.1.0 (Local)
- 开始一次性大范围重构（高频曲线更新逻辑、接收端口模块化、接收管线解耦）
- 统一接收端口抽象与接收事件聚合
- 重写数据接收解析到独立管线
- 曲线更新改为批量消费和定时刷新

## 1.0.0
- 原始稳定基线版本
